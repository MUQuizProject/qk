大会ルール

※Stage 2の問題数について(11/29)

Stage 2で使用する問題数を以下のように変更致します．

(変更前) 各組12問，合計36問
(変更後) 各組15問，合計45問

本大会としましては，Stage 2Bでは，できるだけ多くの問題に解答頂くことで，より得点の差が出やすくなり，勝敗に対する納得感が増すと考えております．この変更はそれを狙ったものです．また，問題数の増加についても，スケジュール上の問題はないと判断し，このように変更させて頂きました．

太字の部分は11/29づけで変更を行った部分です．

Stage 1: 100問筆記クイズ

  • 筆記問題100問と近似値問題1問からなる筆記クイズです．
  • 制限時間は25分です．
  • 100問の問題のうち， 問題番号が5の倍数の問題(5問目，10問目，…，100問目)は「順位決定問題」となります．
  • 順位決定問題は，解答用紙の問題番号が灰色になっています．
  • 以下の基準で順位を決定します．
    1. 全体正解数(100問中)が多い方が上位
    2. (1)が同じ場合，順位決定問題の正解数(20問中)が多い方が上位
    3. (1)(2)がともに同じ場合，近似値問題の誤差が近い方が上位
    4. (1)(2)(3)がすべて同じ場合，大会へのエントリーが早い方が上位
  • 1位〜21位までの参加者は「シード」としてStage 3への進出が保証されるとともに， Stage 2ではStage 2A「早押しボードクイズ」に参加します．
  • 22位以下の参加者はStage 2B「起立ボーナスつきボードクイズ」に参加します．

Stage 2: 早押しボードクイズ & 起立ボーナスつきボードクイズ

  • このステージは2つのクイズを同時進行で行います．
  • シード21名はStage 2A「早押しボードクイズ」に参加します．
  • それ以外の参加者は全員Stage 2B「起立ボーナスつきボードクイズ」に参加します．

Stage 2A: 早押しボードクイズ

  • シード21名を7名ずつ3組にわけて行います．
  • 組分けは以下の通りです．
第1組1位4位7位10位13位16位19位
第2組2位5位8位11位14位17位20位
第3組3位6位9位12位15位18位21位
  • 各組15問の早押しボードクイズを行います．
  • 正解・誤答によるポイント変動は以下の通りです．
正解誤答
ボタン点灯+3pts−3pts
ボード+1pt変動なし
  • 各組ごとに以下の基準で順位を決定します．
    1. 15問終了時のポイントが高い方が上位
    2. (1)が同じ場合，ボタンを点灯させて正解した回数が多い方が上位
    3. (1)(2)が同じ場合，ボタンを点灯させて誤答した回数が少ない方が上位
    4. (1)〜(3)がすべて同じ場合，Stage 1の順位が高い方が上位．
  • 各組ごとに順位に応じて，Stage 3にいて以下のアドバンテージを付与します．
    • 各組1位はStage 3で2ptsからスタート．
    • 各組2位・3位はStage 3で1ptからスタート

Stage 2B: 起立ボーナスつきボードクイズ

  • Stage 2Aと同時進行で行うボードクイズです．
  • 問題はすべてStage 2Aと同一の問題を使用します．
  • 従って，問題数は45問です．
  • 解答は所定の解答用紙に記入してください．
  • なお，早押しボードクイズでボタンが点灯した際，自分の答えに自信があれば，以下の「Stage 2Bの進行の仕方について」に従って起立することができます．
  • 正解・誤答によるポイント変動は以下の通りです．
    正解誤答
    ボタン点灯・起立+3pts−3pts & 次の問題起立不可
    ボタン点灯・通常+1pt変動なし
    スルー+2pts変動なし
    • Stage 2Aのある組の最後の問題で起立誤答した場合は，次の組の最初の問題が起立不可となります．
    • Stage 2Aの3組目の最後の問題は最終問題ですので，そこで起立誤答した場合は-3ptsとなるだけです．
  • 45問終了時に以下の基準で順位をつけます．
    1. ポイントが高い方が上位
    2. (1)が同じ場合，起立正解数の多い方が上位
    3. (1)と(2)が同じ場合，起立誤答数の少ない方が上位
    4. (1)〜(3)がすべて同じ場合，Stage 1の順位が高い方が上位
  • 上位15名がStage 3進出となります．

Stage 2Bの進行の仕方について

Stage 2Bの進行について説明します．

  • まず問題が読まれます．
  • 壇上でStage 2Aに参加している参加者のランプが点灯した場合，以下の流れを取ります．
  • ボタンが点灯した瞬間から3カウントをします．この間に，起立したい参加者は起立してください．
  • 起立したら，解答用紙にある所定の問題番号の「起立」欄に○をつけ，隣同士で相互確認をしてください．
  • 「お座りください」との指示がありましたら座ってください．
  • 「お座りください」の指示から15秒間のうちに解答をお書きください．
  • Stage 2Aの参加者の正誤判定を行っている間に， 得点表示の下部に正解・不正解時の点数が表示されますので， それに従って，以下の3点を書き込んだ上で隣同士で相互確認をしてください．
    • 「正誤」の欄に，正解なら○，誤答なら×を書く．
    • 「点数」の欄に，その問題で得た得点を書く．
    • 「累計」の欄に，その問題までの点数の合計を書く．
  • 壇上のStage 2A参加者のランプが点灯せずスルーとなった場合は，以下の流れをとります．
    • この場合起立ボーナスはなしです． そのまま20秒のシンキングタイムを取りますので，その間に解答をお書きください．
    • 正解を発表後， Stage 2Aの参加者の正誤判定を行っている間に， 得点表示の下部に正解・不正解時の点数が表示されますので，それに従って， 以下の2点を書き込んだ上で隣同士で相互確認をしてくださいてください．
      • 「正誤」の欄に，正解なら○，誤答なら×をつける．
      • 「点数」の欄に，その問題で得た得点をつける．
  • 全問題終了後，点数の合計を計算し，「合計点数」の欄にその合計をお書きください． 合計点も隣同士で相互確認してください．
  • 得点上位者(Stage 2Bの参加者中15位の点数と同点かそれ以上の参加者)の解答用紙を回収し こちらでチェックを行います．

諸注意

  • 解答は，起立した人が座ってから書き始めてください．そうしないと，相互確認の際に自分の答えを相手に見られてしまいます．
  • Stage 2A・2Bともに，問題文の読み直しは行いません．
  • スタッフ側では得点上位者の解答用紙しかチェックをしませんので，得点の管理は周囲の方の協力のもとご自身でお願いします．

Stage 3: 35問早押しクイズ

  • シード21人と，Stage 2Bを勝ち抜けた15人の合計36人を，9人ずつ4組に分けて早押しクイズを行います．
  • 参加者間には予め「優先順位」がついています．優先順位の決定方法は以下の通りです．
    1. シードの方が上位．
    2. シードどうしの順位は
      • Stage 2Aの順位が高い方が上位．
      • それが同じ場合は，Stage 1の順位が高い方が上位．
    3. Stage 2Bからの勝ち上がりの参加者どうしの順位は，Stage 2Bでの順位が高い方が上位．
  • 組み分けは上記「優先順位」に従って以下の通りとします．
第1組1位5位9位13位17位21位25位29位33位
第2組2位6位10位14位18位22位26位30位34位
第3組3位7位11位15位19位23位27位31位35位
第4組4位8位12位16位20位24位28位32位36位
  • 参加者は基本的に0ptsからスタートしますが，Stage 2Aでアドバンテージを付与された参加者は各自そのポイントからスタートとなります．
  • 正解すると 1pt が加算されます.
  • 誤答した場合，それがその参加者にとって n回目の誤答だった場合，次の問題以降，2n問の間解答権を剥奪されます.
  • 従って，最初の誤答は2問，2回目の誤答は4問，3回目の誤答は6問，…と， 剥奪される解答権が増えていきます.
  • 35問経過した時点でその組は終了となります.
  • 35問経過時に，次の基準で順位を決定し，各組上位4名が勝ち抜けとなります.
    1. ポイントが多い方が上位.
    2. (1)が同一の場合，誤答数が少ない方が上位.
    3. (1)と(2)が同じの場合，「優先順位」が高い方が上位．

Stage 4: 30pts先取早押しクイズ

  • Stage 3を勝ち抜けた16名を8名ずつ2組に分けて早押しクイズを行います．
  • 組分けについては以下の「優先順位」に従って，優先順位上位の方から選択していきます．
    1. Stage 3の勝ち抜け順位が高い方が上位
    2. (1)が同一の場合，Stage 1の順位が高い方が上位
  • 参加者は0ptsからスタートします．
  • 正解すると+7ptsされ，誤答すると−4ptsされます．
  • 30pts到達で勝ち抜けとなります．
  • このラウンドでは以下の通り順位を定めます．
    1. 勝ち抜けている方が上位．勝ち抜け者どうしの順位は，先に勝ち抜けた方が上位．
    2. 勝ち抜けていない参加者どうしについては，ポイントの多い方が上位．
    3. (1)(2)ともに同一の場合，上記「優先順位」が高い方が上位．
  • 各組上位4名がSemifinal進出です．
  • 次の条件のうちいずれかが満たされた場合，その組は終了となります．
    1. 4人が勝ち抜けた場合
    2. (1)が満たされておらず，かつ45問終了までに4位と5位の間で逆転不可能となった場合．
      • 「4位と5位の間で逆転不可能」とは，「その時点以降の問題を4位の参加者がすべて誤答しても，順位が逆転しない」ということです．

        例えば残り3問の状況で4位が20pts，5位が6ptsの場合，4位の参加者が3問すべて誤答しても8ptsとなって4位を維持できるので，この時点でその組は終了となります．

    3. (1)(2)ともに満たされず，45問経過した場合．

Semifinal: サバイバル早押しクイズ

  • Stage 4を勝ち抜けた8名が参加します．
  • 各組開始時に，参加者は全員20ptsを持った状態でスタートします.
  • 正解すると，自分のポイントが+1ptされ，同じ組の他の参加者のポイントが−1ptされます.
  • 誤答すると，自分のポイントのみが−1ptされます.
  • ポイントが0pts以下になると，それ以降の解答権を剥奪します. 1
  • 以下の条件のうち少なくとも一つが満たされた場合終了となります．
    • 解答権を持っている参加者が4人以下となる.
    • 60問経過する.
  • 終了時に以下の基準で順位を決定し，上位3名が勝ち抜けとなります．
    1. 終了時のポイントの高い方が上位.
    2. (1)が同一の場合，Stage 4での順位が高い方が上位．
    3. (1)と(2)が同一の場合，Stage 1の順位が高い方が上位．

Final: 配分式早押しクイズ

  • Semifinalを勝ち抜けた3名による早押しクイズです.
  • 参加者は全員0ptsからスタートします.
  • 正解すると正解者のポイントが2pts増えます．
  • 誤答すると，誤答者以外の参加者のポイントが1pt増えます．
  • 20pts以上に到達すると優勝です．
  • 以下の条件のうち少なくとも1つが満たされた場合に終了となります．
    • 優勝が決定する.
    • 50 問経過する.
  • 50 問経過時に優勝者が決まらない場合，次の基準で順位を決定します．
    1. ポイントが多い方が上位．
    2. (1)が同一の場合，正解数が多い方が上位．
    3. (1)と(2)が同一の場合，順位はタイとする．

  1. 注意: 通常「サバイバルクイズ」では，得点が0pts以下になると「失格」し，したがってポイントも残らないとするものが多いです. しかし，本大会では，終了時の判定基準の記述を簡単にするため，得点がマイナスになる場合「解答権剥奪」としてポイントを残し，それ以降もポイントは(他人の正解に伴って)減り続けるものとしています.↩
